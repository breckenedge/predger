# Predger

A simple spending tracker that helps you monitor your expenses. Track what you spend, categorize it, and see your total in real-time.

**[Try it live â†’](https://breckenedge.github.io/predger/)**

## Features

- Add spending items with amounts
- Categorize expenses (Clothing, Food, Fuel, Grooming, Health, Hobby)
- See your running total
- Delete items you no longer need
- Reset all data when starting fresh
- Works offline as a Progressive Web App
- All data stays on your device (localStorage)

## Getting Started

```sh
npm start
```

Then open http://localhost:8080 in your browser.
